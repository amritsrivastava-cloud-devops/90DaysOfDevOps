## ðŸ”¹ Network Interfaces & IP Configuration

| Command | Explanation |
|------|------------|
| `ifconfig` | Show or configure network interfaces (legacy) |
| `ifconfig eth0 up` | Activate `eth0` interface |
| `ifconfig eth0 down` | Deactivate `eth0` interface |
| `ip addr` | Show or manage IP addresses |
| `ip addr show` | Display interfaces and assigned IPs |
| `ip addr add 192.168.1.1/24 dev eth0` | Assign IP address to interface |
| `ip link` | Manage network interfaces |
| `ip link show` | Show interface status |
| `ip link set eth0 down` | Disable network interface |

## ðŸ”¹ Routing & Gateway Management

| Command | Explanation |
|------|------------|
| `ip route` | Show or manage routing table |
| `ip route show` | Display routing table |
| `ip route add default via 192.168.1.1` | Add default gateway |
| `route` | Show or manipulate routing table (legacy) |
| `route -n` | Show routing table without DNS resolution |
| `route add default gw 192.168.1.1` | Add default route |
| `route flush` | Remove all routing entries |
